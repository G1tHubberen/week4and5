Reflections

Initially, adding click handlers to divs for individual console outputs is straightforward, thanks to unique ids.
However, as divs multiply, efficiency diminishes. Leveraging event bubbling within an
outer parent div simplifies handling, creating concise code and improving scalability.
Outputting results to an empty paragraph tag enhances user interaction through the innerText property, enhancing accessibility.
Event bubbling efficiently directs events to the nearest event listener, as seen with red divs within an outer container.
Utilizing evt.target.id, the specific div id responsible for the event is easily identified,
highlighting the power of event bubbling in managing complex interactions.
