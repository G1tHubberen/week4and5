    Event Bubbling:

    We use event bubbling to handle clicks on the SVG map. Instead of adding many click listeners to each country, we have one listener on the map itself. When a country is clicked, the event bubbles up to the map, and we can determine which country was clicked.

    DOM Manipulation:

    modify the web page content using JavaScript. When a country is clicked, we use DOM manipulation to update the "info" div with details about the selected country. This includes its name, capital, region, and more.

    Fetch API:

    We fetch country data from a REST API when a country is clicked. We construct the API URL based on the clicked country's ISO-country code. After receiving the API response, we extract JSON data and use it to populate the "info" div with relevant country information.
